# Hack Content

Each hack has its own folder:

- `meta.json`: short metadata and structured fields
- `wiki.md`: optional long-form markdown article

Example:

```text
content/hacks/copy-0/
  meta.json
  wiki.md
```

`meta.json` drives the homepage cards and generated quick-reference sections. `wiki.md` is where long-form content should live when writing detailed pages.
