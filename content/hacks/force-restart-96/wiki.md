## Overview

Force Restart is the emergency reboot path for a Mac that is no longer responding normally. It is not a convenience shortcut. It is a last-resort recovery action when the system is frozen enough that ordinary Quit, Force Quit, or Restart options are not available.

## How to use it

1. Try normal recovery paths first if the Mac still responds at all.
2. Use the hardware shortcut or button combination your Mac supports for forced restart.
3. Wait for the machine to reboot.
4. Check unsaved work and app recovery state after login.

## Why it matters

Sometimes the system is stuck hard enough that there is no cleaner option. In that situation, a forced restart can be the fastest way back to a usable machine.

## Caveats

- Unsaved data can be lost.
- File corruption risk is higher than with a normal restart.
- Use it only when the Mac is genuinely unresponsive, not merely annoying.
