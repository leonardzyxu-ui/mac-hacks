## Overview

`Cmd + Shift + G` opens Finder's Go to Folder sheet, which lets you jump directly to a path instead of drilling through the interface one click at a time. It is one of the fastest ways to reach nested, hidden, or rarely visited locations on a Mac.

## How to use it

1. Focus Finder.
2. Press `Cmd + Shift + G`.
3. Type or paste a path such as `~/Library`, `/Applications`, or a project directory.
4. Press Return to jump there.

## Why it matters

Go to Folder is the bridge between graphical file browsing and path-based navigation. It is ideal when you know where something lives but do not want to navigate there manually, especially for hidden folders or deep directory structures.

## Caveats

- The path must be valid, so a typo sends you nowhere useful.
- `~` works for your home directory, which is often the most convenient shorthand.
- This is faster than clicking around, but only if you already know the location you are aiming for.
