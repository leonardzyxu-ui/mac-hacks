## Overview

This Finder preference tweak hides all Desktop icons without deleting any of the underlying files. It is a presentation change, not a data-loss event, and it is useful when the Desktop itself is visually noisy.

## How to use it

1. Run the Desktop-visibility command in Terminal.
2. Let Finder restart.
3. Confirm that the Desktop is now visually empty.
4. Keep working normally; the files still exist where they always did.

## Why it matters

Hiding Desktop icons is useful for screen sharing, clean screenshots, presentations, and people who like the Desktop as empty visual space rather than as a working folder.

## Caveats

- The files are still on the Desktop even though you cannot see them.
- If you forget the tweak is enabled, it can feel like the Desktop files vanished.
- Finder restart is expected as part of the change.
