## Overview

Quick Look lets you preview a file without fully opening it in its main app. In Finder, select a file and press Spacebar, and macOS shows a fast preview overlay. It is one of the best file-browsing tricks on the Mac because it reduces the cost of checking what something is before you commit to opening it.

## How to use it

1. Open Finder and highlight a file.
2. Press Spacebar.
3. Read, inspect, or scrub through the preview.
4. Press Spacebar again or Escape to close it.

## Why it matters

Quick Look is ideal for images, PDFs, documents, videos, and many common file formats. It speeds up triage in messy folders, helps you verify attachments before sending them, and lets you scan through files without opening a dozen apps.

## Caveats

- It depends on Finder selection, so nothing happens if no file is highlighted.
- Some file types preview beautifully; others only show limited metadata.
- If you need full editing controls, Quick Look is only the first check, not the final workspace.
