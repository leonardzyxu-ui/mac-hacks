## Overview

In Finder, pressing Return on a selected file enters rename mode instead of opening the file. That is a very Mac-like behavior, and it is useful because it keeps file naming fast and keyboard-driven once you understand the rule.

## How to use it

1. Select the file or folder in Finder.
2. Press Return.
3. Type the new name.
4. Press Return again to confirm.

## Why it matters

Renaming is a constant housekeeping task. Doing it from the keyboard is faster than right-clicking, opening menus, or pausing to click the filename carefully. It is especially useful while cleaning downloads, organizing screenshots, or naming project folders.

## Caveats

- Return renames; `Cmd + O` or a double-click opens.
- Be careful with file extensions if Finder is showing them.
- If the item is currently locked or permissions are restricted, Finder may refuse the rename.
