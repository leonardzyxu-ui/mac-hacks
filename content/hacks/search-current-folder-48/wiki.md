## Overview

`Cmd + F` in Finder starts a search, but the important move is switching the search scope to the current folder instead of the entire Mac. That turns Finder search from a broad system hunt into a targeted project-folder tool.

## How to use it

1. Open the folder you actually want to search.
2. Press `Cmd + F`.
3. Set the scope to the current folder near the top of the Finder window.
4. Add the search term or extra rules you need.

## Why it matters

Folder-scoped search is useful when a project directory is large enough to be messy but still specific enough that you do not want matches from the rest of the machine. It is especially good for source folders, research folders, download dumps, and media collections.

## Caveats

- Finder may default to a wider search scope if you do not change it.
- Search quality depends on file names, metadata, and Spotlight indexing.
- If you are already in the wrong folder, the narrowed search will be very precise and still completely unhelpful.
