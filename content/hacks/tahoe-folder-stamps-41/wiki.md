## Overview

Tahoe Folder Stamps are a newer Finder customization that lets folders carry a more distinctive appearance, such as color tinting or a symbolic marker. The point is visual differentiation: folders become easier to recognize without reading every name.

## How to use it

1. Open Finder on a Tahoe system that supports the feature.
2. Open the folder appearance controls.
3. Apply the visual stamp, tint, or style you want.
4. Use the result consistently so the appearance means something.

## Why it matters

Visual folder identity can speed up recognition in crowded directories, shared demos, and creative workflows where visual categories are easier to scan than plain text names.

## Caveats

- This is presentation, not structure.
- Too many decorative variations can make the file system feel noisy.
- Because it is Tahoe-specific, do not expect the feature on older macOS versions.
