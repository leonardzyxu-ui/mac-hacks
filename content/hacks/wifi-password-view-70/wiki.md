## Overview

macOS can reveal and copy the password for a saved Wi-Fi network from system settings. This is a practical sharing feature: the value is not that you can inspect the password for fun, but that you can hand it to another device or person without guessing or walking to the router.

## How to use it

1. Open Wi-Fi settings.
2. Find the saved network you care about.
3. Use the network controls to reveal or copy the password.
4. Share it securely with the person or device that needs it.

## Why it matters

This saves time whenever you need to onboard a device, help someone else join the same network, or verify that a remembered password is actually the one in use.

## Caveats

- Revealing a password is a security-sensitive action.
- Device authentication may be required before macOS shows it.
- Share the password intentionally; do not treat visibility as a reason to spread it casually.
