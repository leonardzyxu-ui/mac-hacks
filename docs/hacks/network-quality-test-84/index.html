What this hack does

Network Quality Test is a terminal-driven macOS hack for Terminal. Runs Apple's built-in internet speed and responsiveness test. In isolation that may sound small, but this kind of action is exactly the sort of thing that shapes how smooth or clumsy a normal day on a Mac feels.

This hack is tagged High and medium risk. This is not usually the kind of hack that damages a machine by itself, but it can still affect how macOS behaves in ways that surprise you later. A short pause before you confirm the change is usually enough to keep it productive instead of distracting.
