Live Speech

System Settings > Accessibility > Live Speech. Type to have your Mac speak for you during FaceTime calls.

SystemSettings HacksLevel: MediumMethod: SettingsRisk: Low

Overview

This hack is a settings tweak for System.

Type to have your Mac speak for you during FaceTime calls.

Common uses
  • Use voice/assistive features
  • Improve ergonomics and readability
  • Tune macOS behavior to your workflow
  • Find hidden toggles that save time
Troubleshooting
  • Use System Settings search to find the right panel quickly.
  • Some settings may be hidden or locked by device management (MDM).

Quick reference

Reference
System Settings > Accessibility > Live Speech

How to use

  1. System Settings > Accessibility > Live Speech
  2. Type to have your Mac speak for you during FaceTime calls.
  3. If you want to undo it later, return to the same setting and switch it back.
