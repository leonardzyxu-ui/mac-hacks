Personal Voice

Settings > Accessibility > Personal Voice. Train the Mac to speak in YOUR own voice using AI.

SystemSettings HacksLevel: HighMethod: SettingsRisk: LowTahoe

Overview

This hack is a settings tweak for System.

Train the Mac to speak in YOUR own voice using AI.

Prerequisites
  • Access to System Settings
Common uses
  • Use voice/assistive features
  • Improve ergonomics and readability
  • Tune macOS behavior to your workflow
  • Find hidden toggles that save time

Quick reference

Reference
Settings > Accessibility > Personal Voice

How to use

  1. Open System Settings (or the relevant app).
  2. Settings > Accessibility > Personal Voice
  3. Train the Mac to speak in YOUR own voice using AI.
  4. If you want to undo it later, return to the same setting and switch it back.
‚úì How to verify
  • The setting toggle should reflect your new choice
  • Changes may take effect immediately or after closing System Settings
Compatibility
Requires: macOS 26.0+
Requires macOS Tahoe (26.0) or later

Troubleshooting

  • Use System Settings search to find the right panel quickly.
  • Some settings may be hidden or locked by device management (MDM).
