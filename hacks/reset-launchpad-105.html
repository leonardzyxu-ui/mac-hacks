Reset Launchpad

'defaults write com.apple.dock ResetLaunchPad -bool true; killall Dock'. Resets Launchpad to factory layout.

TerminalTerminal HacksLevel: ProfessionalMethod: TerminalRisk: Medium

Overview

This hack is a Terminal command for Terminal.

Resets Launchpad to factory layout.

Prerequisites
  • Terminal app (built into macOS)
Common uses
  • Advanced customization and power-user tweaks
  • Troubleshoot system behavior with commands
  • Reset stuck UI components
  • Boot into a safe state to debug issues
  • Switch apps/windows quickly
  • Split-screen and multitask efficiently

Quick reference

Command
defaults write com.apple.dock ResetLaunchPad -bool true; killall Dock

How to use

  1. Open Terminal.
  2. Run: defaults write com.apple.dock ResetLaunchPad -bool true; killall Dock
  3. Resets Launchpad to factory layout.

Troubleshooting

  • Read the command fully before running it; avoid `sudo` unless necessary.
  • If nothing changes, you may need to quit/reopen the relevant app for the setting to apply.
