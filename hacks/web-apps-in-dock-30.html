Web Apps in Dock

Safari > File > Add to Dock. Turns any website into a standalone app icon in your Dock.

SafariOtherLevel: HighMethod: UIRisk: Low

Overview

This hack is a workflow tip for Safari.

Turns any website into a standalone app icon in your Dock.

Common uses
  • Manage tabs and history faster
  • Browse with less distraction (e.g., private windows)
  • Move text between apps quickly
  • Copy files in Finder and paste into another folder
  • Switch apps/windows quickly
  • Split-screen and multitask efficiently

Quick reference

Reference
Safari > File > Add to Dock

How to use

  1. Safari > File > Add to Dock
  2. Turns any website into a standalone app icon in your Dock.
  3. Look for the same action in the menu bar if you prefer clicking.
